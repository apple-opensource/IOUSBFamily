Accessing USB devices from User Space

Introduction

This document describes the details of the programming interface for accessing USB devices and USB interfaces from code running in User Space. This document assumes that you have a basic understanding of the material contained in the document Accessing Hardware From Applications (available at http://developer.apple.com/techpubs/macosx/Kernel/kernel.html.) For definitions of I/O Kit terms used in this document, such as matching dictionary, family, and driver, see the overview of I/O Kit terms and concepts in the "I/O Kit Hardware Access" chapter of Accessing Hardware From Applications.

This document also assumes that you have read the document Working with USB Device Interfaces (available at http://developer.apple.com/techpubs/macosx/Darwin/IOKit/DeviceInterfaces/USBBook/index.html). Please review that document before trying to use this reference.

All of the information described in this document is contained in the header file IOUSBLib.h found at /System/Library/Frameworks/IOKit.framework/Headers/usb/IOUSBLib.h.
